Colophon

This website is built with Eleventy, stored on GitHub in a public repository and hosted on Vercel.

I want to keep it simple and fast, so I don't use any JavaScript frameworks and only a few CSS styles. The website is designed to be usable without JavaScript and has no images. I also use the default system fonts, so you can choose whatever font you like.

The site is also part of the IndieWeb. You can send a webmention to every page.
